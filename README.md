# Gambling

## Instruções iniciais
Antes de começarmos, certifique-se que tenha instalado o **python** e **virtualenv**. Abra o Terminal e crie uma virtualenv chamada **env**.
No Windows:
<pre>
virtualenv env
 . env env/Script/activate
</pre>
No Linux:
<pre>
virtualenv env
 . env env/bin/activate
</pre>
